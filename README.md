# Student Management System (Java)

## 📌 Description
A console-based Java application to manage student records using CRUD operations.

## 🚀 Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- File Handling (Data Persistence)

## 🛠️ Technologies Used
- Java
- ArrayList
- File Handling

## ▶️ How to Run
1. Open project in NetBeans
2. Run Main.java
3. Use menu options

## 📂 Project Structure
- Student.java
- StudentService.java
- Main.java
